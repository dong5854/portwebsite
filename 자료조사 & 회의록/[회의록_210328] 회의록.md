## 작년 수상 작품 주요 기능
- 선박 입출항 실시간 정보 시각화
- 울산항만이 관리하는 부두 상태를 실시간 Chart로 시각화
- 정박 선박의 정보 실시간 제공
- 주유품목 물동량 예측
- 부두별 하역 능력 평가
- 부두별 접안 능력 평가

## 차별화된 기능 구현의 방향성
- 항만 관계자 의사 결정 지원 대시보드 구축

## 구현 예정 세부 기능
- 키바나(kibana) or 태블로 활용해 시각화 지원
- 대시보드 이용 대상 확장
- 물동량 예측 모델 구현
- 선박 배치 모델 구현
- 선박 대기 시간 예측 모델 구현
- 하역 능력에 새로운 지표 창출
